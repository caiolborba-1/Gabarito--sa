O QUE É UM GRUD?


O CRUD É UM ACRÔNIMO PARA AS OPERAÇÕES FUNDAMENTAIS EM QUALQUER SISTEMA, QUE INTERAGE COM O BANCO DE DADOS: CREATE, READ, UPDATE, DELETE.
ESSAS 4 OPERAÇÕES PERMITEM QUE OS USUÁRIOS POSSAM CRIAR REGISTROS NOVOS, LER OU RECUPERAR DADOS EXISTENTES, ATUALIZAR INFORMAÇÕES PREVIAMENTE ARMAZENADAS, E FINALMENTE EXCLUIR REGISTROS QUANDO NECESSÁRIOS
